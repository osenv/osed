# osenv.org Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build v1 of osenv.org — an umbrella site for open-source environmental projects with OSED as the flagship — as a new `osenv` repo: a vision-forward authored home/about/projects + OSED's two-track guide rendered from content synced (never copied) from the `osed` repo, on a Flexoki earth/foliage/water palette.

**Architecture:** Astro + Tailwind + React islands, static output to Cloudflare Pages. The `osed` repo is a pinned git submodule at `vendor/osed`; a prebuild script (`scripts/sync-guide.mjs`) copies `docs/guide/**` into an Astro content collection and rewrites the guide's relative links (intra-guide → site routes; escapes → GitHub-at-SHA; external → untouched; unknown → build fails). Content freshness is pin + automated bump PR. The guide's voice and disclaimer stay intact; osenv frames, never re-authors.

**Tech Stack:** Astro, Tailwind CSS, React (islands), MDX, Vitest (sync-script tests), GitHub Actions, Cloudflare Pages. Node 20+.

**Spec:** `docs/specs/osenv-website-design.md` (authored in the osed repo; carried into osenv in Task 1).

- [ ] **Step 4: Implement `src/lib/guide-links.mjs`**

```js
import path from "node:path";

const GUIDE_PREFIX = "docs/guide/";
const ROUTE_BASE = "/projects/osed/guide";

/** Map a repo-relative guide path (under docs/guide/) to its site route. */
export function toSiteRoute(repoPath) {
  if (!repoPath.startsWith(GUIDE_PREFIX)) {
    throw new Error(`not a guide path: ${repoPath}`);
  }
  let rel = repoPath.slice(GUIDE_PREFIX.length).replace(/\.md$/, "");
  if (rel === "README") return ROUTE_BASE;
  return `${ROUTE_BASE}/${rel}`;
}

/**
 * Rewrite one markdown link from a guide file.
 * @param {{href:string, fileRepoPath:string, sha:string, org:string, exists:(p:string)=>boolean}} a
 * @returns {string} the rewritten href
 */
export function rewriteLink({ href, fileRepoPath, sha, org, exists }) {
  if (/^(https?:|mailto:)/i.test(href)) return href;
  const [rawPath, hash] = splitHash(href);
  if (rawPath === "") return href; // pure same-page anchor

  const dir = path.posix.dirname(fileRepoPath);
  const target = path.posix.normalize(path.posix.join(dir, rawPath));

  if (!exists(target)) {
    throw new Error(`unresolved link "${href}" in ${fileRepoPath} (resolved to ${target})`);
  }
  const suffix = hash ? `#${hash}` : "";
  if (target.startsWith(GUIDE_PREFIX)) return toSiteRoute(target) + suffix;
  return `https://github.com/${org}/osed/blob/${sha}/${target}${suffix}`;
}

function splitHash(href) {
  const i = href.indexOf("#");
  if (i === -1) return [href, ""];
  return [href.slice(0, i), href.slice(i + 1)];
}
```

- [ ] **Step 2: Write `scripts/sync-guide.mjs`**

```js
import fs from "node:fs";
import path from "node:path";
import { execSync } from "node:child_process";
import { rewriteLink } from "../src/lib/guide-links.mjs";

const ROOT = path.resolve(process.cwd());
const VENDOR = path.join(ROOT, "vendor", "osed");
const GUIDE_SRC = path.join(VENDOR, "docs", "guide");
const OUT = path.join(ROOT, "src", "content", "guide");
const ORG = process.env.OSED_ORG || "REPLACE_ORG"; // set in CI / .env

const SHA = execSync("git rev-parse HEAD", { cwd: VENDOR }).toString().trim();

// Existence predicate against the pinned submodule working tree.
const exists = (repoRel) => fs.existsSync(path.join(VENDOR, repoRel));

// Map a directory name to the content "branch" field.
const branchOf = (relDir) =>
  relDir === "" ? "root" : relDir.split("/")[0];

const LINK_RE = /\]\(([^)]+)\)/g;

function* walk(dir) {
  for (const e of fs.readdirSync(dir, { withFileTypes: true })) {
    const full = path.join(dir, e.name);
    if (e.isDirectory()) yield* walk(full);
    else if (e.name.endsWith(".md")) yield full;
  }
}

function titleFrom(md, fallback) {
  const m = md.match(/^#\s+(.+)$/m);
  return m ? m[1].trim() : fallback;
}

fs.rmSync(OUT, { recursive: true, force: true });
fs.mkdirSync(OUT, { recursive: true });

let count = 0;
for (const file of walk(GUIDE_SRC)) {
  const relFromGuide = path.relative(GUIDE_SRC, file).split(path.sep).join("/");
  const fileRepoPath = `docs/guide/${relFromGuide}`;
  const relDir = path.posix.dirname(relFromGuide) === "." ? "" : path.posix.dirname(relFromGuide);
  let body = fs.readFileSync(file, "utf8");

  body = body.replace(LINK_RE, (whole, href) => {
    const out = rewriteLink({ href, fileRepoPath, sha: SHA, org: ORG, exists });
    return `](${out})`;
  });

  const title = titleFrom(body, relFromGuide);
  const branch = branchOf(relDir);
  const frontmatter = `---\ntitle: ${JSON.stringify(title)}\nbranch: ${branch}\n---\n\n`;

  const outPath = path.join(OUT, relFromGuide);
  fs.mkdirSync(path.dirname(outPath), { recursive: true });
  fs.writeFileSync(outPath, frontmatter + body);
  count++;
}
console.log(`synced ${count} guide files from osed@${SHA.slice(0, 7)} (org=${ORG})`);
if (count === 0) throw new Error("sync produced 0 files — submodule missing? run with --recurse-submodules");
```

## Task 6: CI — build with submodules, sync tests, link integrity, bump PR

- [ ] **Step 1: Write `.github/workflows/ci.yml`**

```yaml
name: ci
on:
  push: { branches: [main] }
  pull_request:
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with: { submodules: recursive }
      - uses: actions/setup-node@v4
        with: { node-version: "20", cache: "npm" }
      - run: npm ci
      - run: npm test
      - run: npm run build
        env:
          OSED_ORG: ${{ github.repository_owner }}
```

- [ ] **Step 2: Write `.github/workflows/bump-osed.yml`** (scheduled + manual + dispatch → reviewable PR)

```yaml
name: bump-osed-guide
on:
  schedule: [{ cron: "0 7 * * *" }]   # nightly fallback
  workflow_dispatch:
  repository_dispatch: { types: [osed-guide-changed] }
permissions:
  contents: write
  pull-requests: write
jobs:
  bump:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with: { submodules: recursive }
      - name: Advance the osed submodule to latest main
        run: |
          git -C vendor/osed fetch origin main
          git -C vendor/osed checkout origin/main
          echo "OSED_SHA=$(git -C vendor/osed rev-parse --short HEAD)" >> "$GITHUB_ENV"
      - name: Open a PR if the pin moved
        uses: peter-evans/create-pull-request@v6
        with:
          branch: bump/osed-guide
          title: "chore: bump osed guide to ${{ env.OSED_SHA }}"
          body: "Automated submodule bump. Review the guide diff before merging; the site rebuilds on merge."
          commit-message: "chore: bump osed guide to ${{ env.OSED_SHA }}"
          add-paths: vendor/osed
```

- [ ] **Step 3: Verify the workflows are valid YAML locally**

Expected: prints `ci.yml ok` (the checkout MUST recurse submodules or the build has no guide content).
